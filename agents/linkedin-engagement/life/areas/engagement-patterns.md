# LinkedIn Engagement Patterns — Akira Agent

> Self-updating memory file. Read at the start of every heartbeat; weekly synthesis updates it based on what drove replies.
>
> **Last updated:** 2026-04-15 (seed defaults)

## Daily volume target

- 5 actions/day (comments + reactions). Actions count only on ICP posts.
- Rough split per day: 3 comments + 2 reactions, adjust based on what's landing.

## Topic allow-list (auto-action without approval)

Comment or react without approval when the post is about:

- Restaurant or hotel operations (reservations, no-shows, shift scheduling, guest experience).
- AI / automation in hospitality or hospitality adjacent.
- Nordic SMB technology adoption.
- Labor costs / admin staff productivity.

All other topics → **draft and escalate to Head of Marketing for approval.**

## Topic block-list (never comment or react)

- Politics (any country).
- Religion.
- Unrelated personal matters (weddings, funerals, memorials unless of an Akira connection).
- Competitor promotional posts (observe silently).
- Any post in Norwegian.

## Comment language rules

- Match the post's language: Swedish, English, or French.
- Never Norwegian.
- If the post is in a language not in that list (German, Italian, etc.), default to English.

## Comment style (seed)

- One idea per comment. 1–3 sentences. No emoji spam. Zero "Great post!" noise.
- Add value: a specific data point, a question that advances the thread, or a concrete example.
- Never pitch Akira in a comment. That's a DM or a profile click.
- Never use em-dashes or AI-obvious phrasing patterns. Write like Jules would write.

## DM cadence

- Only to ICP prospects already engaged with at least once on a public post (comment or reaction).
- Language matches profile.
- **All DMs are draft-first → Jules approves → send.** No auto-send.
- Max 10 DMs/week (platform-safe).
