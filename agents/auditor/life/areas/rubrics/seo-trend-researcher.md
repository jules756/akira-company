# Rubric — SEO Trend Researcher

Scored daily by Auditor. Each dimension 0–5.

### 1. Daily trend scan completeness (0–5)
- **5** — Every business day, scan covered all four source classes: (a) industry blogs (Search Engine Journal, Ahrefs, SEMrush, Moz), (b) Google Search Central announcements, (c) AEO/GEO chatter (Twitter/X, LinkedIn, AI search platform changelogs), (d) competitor SEO moves. Findings logged in `trend-log.md`.
- **3** — 2–3 source classes covered.
- **1** — Single source.
- **0** — Skipped.

### 2. Actionable insight conversion (0–5)
- **5** — Every meaningful trend produces one of: (a) new keyword target added to roadmap, (b) updated pattern in `seo-playbook.md`, (c) new article brief proposal, (d) deprecation flag on something we were doing. No "interesting article" links without action.
- **3** — Most insights actionable.
- **1** — Mostly link dumps.
- **0** — No conversion.

### 3. Playbook freshness (0–5)
- **5** — `seo-playbook.md` updated weekly with new tactics. `keyword-roadmap.md` extended with trend-driven candidates.
- **3** — Updates monthly.
- **1** — Stale playbook.
- **0** — Never updates.

### 4. Speed on Google updates (0–5)
- **5** — Major Google Search update or AI search platform change → action proposal to SEO + AEO Specialist within 48h. We don't get caught flat-footed by core updates.
- **3** — Notice within a week.
- **1** — Notice after impact.
- **0** — Misses entirely.

### 5. Coordination with execution team (0–5)
- **5** — Trend insights handed cleanly to SEO + AEO Specialist (for keyword roadmap), Web Writer (for content angles), Web Operations (for technical SEO updates). Each handoff has the *what + why + suggested action*.
- **3** — Some handoffs vague.
- **1** — Insights stay in trend-log without team distribution.
- **0** — Working in isolation.
