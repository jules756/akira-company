# Rubric — Web Writer

Scored daily by Auditor. Each dimension 0–5.

### 1. Publishing cadence (0–5)
- **5** — 1–2 articles published per business day on akira-agent.com/blog, on the SEO + AEO Specialist's roadmap. No backlog of un-shipped briefs >3 days old.
- **3** — 4–7 articles/week (vs. target 5–10).
- **1** — <4 articles/week.
- **0** — Stalled.

### 2. Brief adherence (0–5)
- **5** — Every article matches its SEO + AEO brief: target query covered, search intent satisfied, outline followed, internal links placed, AEO answer block present, schema noted for Web Operations.
- **3** — Mostly matches; occasional gaps.
- **1** — Drifts from briefs.
- **0** — Briefs ignored.

### 3. Brand voice (0–5)
- **5** — Every article passes brand-voice-enforcement. No "delve / robust / seamless / not just X, it's Y" patterns. Specific Nordic-hospitality detail in every piece.
- **3** — Mostly clean, occasional drift.
- **1** — Generic AI-sounding output.
- **0** — Off-voice.

### 4. Case-study quality (0–5)
- **5** — Every Production client has a case study within 30 days of going live (anonymized if needed). Case studies hit the Cold-to-Cash pattern: specific number + specific story + specific lesson.
- **3** — Most Production clients covered, some delayed.
- **1** — Case studies vague or missing.
- **0** — No case studies.

### 5. Coordination + handoff (0–5)
- **5** — Every shipped article gets passed to Web Operations with: schema metadata, target URL, internal-link map, OG image request to Web Designer, distribution note for Content Writer (LinkedIn).
- **3** — Most handoffs clean.
- **1** — Web Operations chases for metadata.
- **0** — Articles published without handoff trail.
