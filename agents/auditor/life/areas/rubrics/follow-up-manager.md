# Rubric — Follow-Up Manager

> Scored daily by the Auditor. Each dimension 0–5.

### 1. Cadence adherence (0–5)

- **5** — No active lead has gone silent past its `Recontact On` date + grace window (≤1 day). Cadence rules in `life/areas/follow-up-patterns.md` applied consistently.
- **3** — ≤2 leads overdue by >1 day; cadence rules applied but with occasional misses.
- **1** — >5 leads overdue.
- **0** — Cadence abandoned; leads silent >7 days past Recontact On.

### 2. Outreach Log completeness (0–5)

- **5** — Every reply, outbound email, or DM creates a matching Outreach Log row with Channel, Date, Outcome, Notes, Recontact On.
- **3** — Most touches logged; notes sometimes thin.
- **1** — Touches frequently missing from the log.
- **0** — Gmail activity not reflected in the log.

### 3. Reply handling quality (0–5)

- **5** — Replies matched to the correct CRM record within one heartbeat; Outcome set correctly; Head of Sales pinged on any `Meeting booked` / `Not interested` / `Feedback meeting` / stage-relevant event.
- **3** — Matched correctly but sometimes late; escalations incomplete.
- **1** — Mismatches; replies attached to wrong Company.
- **0** — Replies ignored.

### 4. Memory / learning behavior (0–5)

- **5** — Weekly synthesis updates `follow-up-patterns.md` with outcome-driven learnings ("Waiting 5 days for fine-dining leads outperforms 3 days by X% reply rate in last 4 weeks"). Rules adapt.
- **3** — Synthesis runs but doesn't lead to rule updates.
- **1** — No synthesis.
- **0** — No memory writes; treats every week identically.

### 5. Demo coordination (0–5)

- **5** — When a reply indicates interest in a demo, books a calendar event, sends invite, creates a prep brief, and pings Head of Sales — all in one heartbeat.
- **3** — Books the demo but prep brief missing.
- **1** — Replies noted but no demo action.
- **0** — Demo-intent replies go cold.
