# SEO + AEO Growth

The strategic engine for organic discovery. Classical SEO (Google rankings) AND AEO/GEO (citations in ChatGPT, Perplexity, Claude, Gemini answers). Daily SEO landscape scouting, weekly Search Console deep analysis, AI search citation tracking, content roadmap building. The ammunition that feeds the daily publishing engine on `web-publishing`.

## Scope

- **Keyword + AEO roadmap** — always ≥4 weeks of article briefs queued for Web Writer.
- **AI citation tracking** — weekly probe of ChatGPT / Perplexity / Claude / Gemini against tracked queries; gaps → new briefs.
- **SEO trend scouting** — daily scan across 4 source classes (industry blogs, Google Search Central, AEO/GEO platforms, competitor moves). 48h SLA on major Google or LLM platform updates.
- **Article briefs** — every blog post starts here, with target query, search intent, AEO answer block, internal-link map, schema spec.
- **Search Console deep analysis** — weekly Friday tech-SEO health check (Web Operations does daily monitoring; SEO + AEO Specialist does weekly strategic analysis).
- **Playbook + roadmap maintenance** — `seo-playbook.md`, `keyword-roadmap.md`, `aeo-citation-log.md`, `trend-log.md`, `source-list.md` all live + updated continuously.
- **Backlink strategy** — identify partner sites + Nordic hospitality publications worth pursuing; coordinate outreach via Content Writer.

## Primary agents

- **SEO + AEO Specialist** — owns the keyword + AEO roadmap; briefs every Web Writer article; weekly tech health check.
- **SEO Trend Researcher** — owns daily landscape scouting + Monday weekly synthesis; co-maintains `seo-playbook.md`.

Both report to Head of Marketing. Coordinate with Web Writer (articles), Web Operations (technical SEO + GSC implementation), Content Writer (backlink outreach), Lead Magnet Creator (magnet SEO angles).
