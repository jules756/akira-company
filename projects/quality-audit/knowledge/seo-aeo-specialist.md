# Rubric — SEO + AEO Specialist

Scored daily by Auditor. Each dimension 0–5.

### 1. Keyword + AEO roadmap freshness (0–5)
- **5** — `seo-playbook.md` + `keyword-roadmap.md` updated weekly with fresh queries (classical SEO + AEO/GEO targets). Roadmap has 4+ weeks of articles ahead of Web Writer.
- **3** — Roadmap exists but ages slowly.
- **1** — One-off keyword work, no system.
- **0** — No roadmap.

### 2. Article brief quality (0–5)
- **5** — Every article Web Writer ships has an SEO + AEO brief: target query, search intent, outline, internal linking targets, AEO angle (citable answer block), schema-markup recommendation.
- **3** — Briefs present but inconsistent.
- **1** — Web Writer fills gaps because briefs are thin.
- **0** — No briefs.

### 3. Technical SEO health (0–5)
- **5** — Sitemap + robots + schema markup current. No 404s on indexed URLs. Core Web Vitals green per page.
- **3** — Mostly healthy, occasional regression.
- **1** — Multiple unresolved technical issues.
- **0** — Site SEO broken.

### 4. AI search citation tracking (0–5)
- **5** — Weekly check: does ChatGPT / Perplexity / Claude / Gemini cite akira-agent.com when asked Nordic-hospitality-AI questions? Wins logged, gaps surfaced as new AEO targets.
- **3** — Tracked but not weekly.
- **1** — Tracked occasionally.
- **0** — Not tracked.

### 5. Coordination quality (0–5)
- **5** — Briefs to Web Writer always 1+ week ahead of publish date. Tags + schema requests to Web Operations clear and unambiguous. Backlink campaigns coordinated with Head of Marketing.
- **3** — Mostly clear, some last-minute briefs.
- **1** — Frequent rework.
- **0** — Web Writer or Web Operations blocked by missing briefs.
